# Delphine de Lapeyrière - Site Web

Site web pour Delphine de Lapeyrière, biographe familiale.

## Technologies

- Next.js 14
- TypeScript
- Tailwind CSS
- React

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

```bash
npm run build
npm start
```
